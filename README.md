# Bröllopsresan

En mobilanpassad, utskrivbar samlingsplats för resplan, bokningar, tider och kostnader.

## Visa lokalt

Öppna `index.html` för en snabb förhandsvisning. Offlinefunktionen kräver att sidan körs via en webbserver eller GitHub Pages.

## Publicera på GitHub Pages

1. Skapa ett GitHub-repository och lägg in filerna.
2. Öppna **Settings → Pages** i repositoryt.
3. Välj **Deploy from a branch**, `main` och `/ (root)`.
4. Öppna sidans adress i Safari på iPhone och välj **Dela → Lägg till på hemskärmen**.

## Utskrift / PDF

Tryck på **Skriv ut** på sidan. På iPhone kan utskriftsförhandsvisningen delas eller sparas som PDF.

## Uppdateringar

Resans innehåll finns tills vidare i `app.js`. Originalarket ligger orört i `data/`. När fler underlag kommer läggs adresser, länkar och exakta tider in i tydliga bokningskort. Känsliga bokningsreferenser och biljettfiler ska inte publiceras på Pages-sidan.
